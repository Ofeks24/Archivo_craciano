# El Archivo Craciano

## Nueva identidad del proyecto

El proyecto deja de presentarse como una plataforma técnica de RAG/documentos IA y pasa a convertirse en:

> Un archivo arcano viviente donde los estudiosos consultan criaturas, grimorios, reliquias, conjuros y conocimientos prohibidos mediante una inteligencia artificial alquímica.

La idea es que el usuario no sienta que usa:

* un panel admin
* una base vectorial
* una IA documental

Sino:

* un archivo mágico
* un bestiario antiguo
* una biblioteca prohibida
* un grimorio inteligente
